# Deep Learning for Sequential Data — RNNs, LSTMs & BERT

## Overview
End-to-end NLP pipeline for 6-class question classification using 
recurrent models and transformer fine-tuning.

## Projects
- **Notebook 1:** Manual RNN from scratch (no built-in layers), 
  raw PyTorch tensor operations
- **Notebook 2:** BaseRNN class supporting vanilla RNN, GRU & LSTM 
  with configurable pooling strategies
- **Notebook 3:** Fine-tuned BERT (bert-base-uncased) with HuggingFace 
  Transformers for sequence classification

## Tech Stack
Python · PyTorch · HuggingFace Transformers · BERT · NumPy · 
Scikit-learn · Google Colab

## Dataset
TREC question classification dataset — 2,000 questions across 
6 categories (ABBR, ENTY, DESC, HUM, LOC, NUM)

## Run
Open the notebooks in Google Colab or Jupyter.
